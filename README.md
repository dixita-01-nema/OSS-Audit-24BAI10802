## About Git

Git is an open-source distributed version control system created by **Linus Torvalds** in 2005. It was developed to support the Linux kernel development after BitKeeper licensing issues.

### Key Features:

* Distributed architecture
* Fast performance
* Branching and merging
* Data integrity using hashing
* Open-source under GPL v2

##  Shell Scripts Description

### 🔹 Script 1 — System Identity Report

This script displays detailed system information including:

* OS distribution
* Kernel version
* CPU and memory details
* Logged-in user information



### 🔹 Script 2 — Package Inspector

* Checks if Git is installed
* Displays version and package details
* Identifies package manager (dpkg/rpm)



### 🔹 Script 3 — Disk & Permission Auditor

* Audits important Linux directories
* Displays permissions, ownership, and size
  


### 🔹 Script 4 — Log File Analyzer

* Reads system log files
* Counts keyword occurrences
* Displays recent matching entries



### 🔹 Script 5 — Manifesto Generator

* Takes user input
* Generates a personalized open-source statement
* Saves it to a file
